# 🌐 Aanchal Pandey — Personal Portfolio Website

A responsive, animated personal portfolio website built to showcase my skills, projects, and achievements as a B.Tech CSE (Cyber Security) student.

## ✨ Features

- **Home** — Animated introduction with typing effect showcasing my roles
- **About** — Brief intro about my background and education
- **Skills** — Python, HTML, CSS, Cyber Security basics, MS Excel
- **Achievements** — Certificates and hackathon wins with a click-to-view certificate viewer
- **Projects** — Featured project cards with live demo/GitHub links and tech stack tags
- **Contact** — Direct links to Email, LinkedIn, and GitHub
- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll animations using ScrollReveal.js
- Typing text animation using Typed.js

## 🛠️ Built With

- **HTML5** — page structure
- **CSS3** — styling, layout, responsiveness
- **JavaScript** — interactivity (menu toggle, scroll spy, lightbox)
- [Typed.js](https://github.com/mattboldt/typed.js/) — typing animation
- [ScrollReveal](https://scrollrevealjs.org/) — scroll-triggered animations
- [Boxicons](https://boxicons.com/) — icon library

## 🚀 Getting Started

### Run locally
1. Clone the repository
   ```bash
   git clone https://github.com/anchal16-cmd/aanchal-portfolio.git
   ```
2. Open the folder in VS Code
3. Right-click `index.html` → **Open with Live Server**
   (or simply double-click `index.html` to open it in your browser)
